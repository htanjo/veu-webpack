# vue-webpack
> Vue.js + webpack project

## Features
- Vue.js 2
- Webpack 2
- Dev server with Hot module replacement
- ES6 using Babel
- ESLint
- Build for production (Minify + Hash filename)

## Workflow
Setup:

```
$ npm install
```

Develop:

```
$ npm start
```

Build for produciton:

```
$ npm run build
```
